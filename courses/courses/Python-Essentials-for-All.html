Page 3: Control Flow

1Ô∏è‚É£ Making Decisions: if, elif, else

Python runs certain code blocks only when conditions are true.

score = 85
if score >= 90:
    print("A grade")
elif score >= 75:
    print("B grade")
else:
    print("C grade")

Indentation matters! Every block under an if or else must be indented.

2Ô∏è‚É£ Loops

Loops let you repeat actions.

for loop ‚Äì iterate over a sequence:

for i in range(3):
    print(i)   # prints 0,1,2

while loop ‚Äì repeat while a condition is true:

count = 0
while count < 3:
    print(count)
    count += 1

3Ô∏è‚É£ Loop Controls

break ‚Äì exit the loop immediately.
continue ‚Äì skip the rest of the current iteration and move to the next.

Example:

for i in range(5):
    if i == 2:
        continue   # skips printing 2
    if i == 4:
        break      # stops loop when i == 4
    print(i)

4Ô∏è‚É£ Boolean Logic Recap

Combine conditions using:

    print("Eligible")

1Ô∏è‚É£ Why Use Functions?

Functions let you group code into reusable blocks.

2Ô∏è‚É£ Defining a Function

3Ô∏è‚É£ Scope of Variables

Local variables exist only inside a function.
Global variables exist outside and can be accessed inside if declared:

4Ô∏è‚É£ Modules

Create your own module by saving a .py file and importing it:

2Ô∏è‚É£ File Handling Basics

Reading and writing files is easy with open().

Reading

with open("data.txt", "r") as f:
    content = f.read()
    print(content)

Writing (overwrite)

with open("data.txt", "w") as f:
    f.write("Hello File!")

Appending (add to existing content)

with open("data.txt", "a") as f:
    f.write("\nMore text")

Mode shortcuts:

  • "r" ‚Äì read (file must exist)
  • "w" ‚Äì write (creates/overwrites)
  • "a" ‚Äì append (creates if not exists)

3Ô∏è‚É£ Practical Tips

  • Always use with open(...) so files close automatically.
  • Use f.readlines() to read line by line for large files.
  • JSON files can be handled with the built-in json module.

  • Python‚Äôs collection types (lists, tuples, dicts) are essential for managing data.
  • Proper file handling is crucial for reading from and writing to data sources.
